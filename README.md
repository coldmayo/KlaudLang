## Features

- Compiler
- Variables
- PEMDAS/Math operations
- Integer support
- Comparison Operators
- Silly Syntax

## How to Use

Make sure you have GCC installed

1. Make a file with the .kld extension
2. run ```make test filename=whatever-your-filename-is.kld```
